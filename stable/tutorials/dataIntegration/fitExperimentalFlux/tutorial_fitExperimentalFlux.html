Quadratically penalise devation from a given (subset of) experimental fluxes

Author(s): Ronan M.T. Fleming, School of Medicine, University of Galway

Reviewer(s):

A given vector of experimentally measured fluxes , may not be consistent with the feasible set of steady state fluxes. Specifically, inconsistent with the set defined by the steady state constraint (Sv=0) as well as the lower and upper bounds on each reactionl, l ≤ v ≤ u. Should this occur, we fit the model to the experimental data, relaxing the constraints on the bounds, that admits a steady state flux v , using the following quadriatic optimisation problem
where p ≥ 0 and q ≥ 0 are non-negative variables that permit relaxation of the lower and upper bound constraints, respectively. This formulation also allows for different weights to be input as parameters to penalise deviation from experimentally measured mean fluxes, with = weightExp, penalise relaxation of lower bounds, with = weightLower, and penalise relaxation of upper bounds, with = weightUpper. For example, if the experimentally measured flux is actually the mean flux, then one could set to be the inverse of the variance on the experimental flux measurements, thereby increasing the penalty on deviation from an experimentally measured mean flux where the variance is lower. Certain lower or upper bounds might not be realistic to be relaxed, e.g., an essential amino acid can always be taken up but never secreted, therefore the upper bound on the corresponding exchange reaction must be zero. The code below also works if there are additional coupling constraints, , but these are not relaxed. This approach was developed and applied within [1].
